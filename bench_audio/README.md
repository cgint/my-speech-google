# bench_audio

Small, repeatable audio samples for comparing transcription approaches.

All `*_16k_mono_pcm16.wav` files are:
- PCM16 (signed 16-bit)
- 16,000 Hz
- mono

Generated on macOS via `say` (to AIFF) and converted with `ffmpeg`.

See `manifest.tsv` for the intended text and language hints.
